## Quando usar esta habilidade

Para escrever comunicações internas, use esta habilidade para:

- Atualizações 3P (Progresso, Planos, Problemas)
- Boletins informativos da empresa
- Respostas de FAQ
- Relatórios de status
- Atualizações de liderança
- Atualizações de projetos
- Relatórios de incidentes

## Como usar esta habilidade

Para escrever qualquer comunicação interna:

1. **Identifique o tipo de comunicação** a partir da solicitação
2. **Carregue o arquivo de diretriz apropriado** do diretório `examples/`:
   - `examples/3p-updates.md` - Para atualizações de equipe Progresso/Planos/Problemas
   - `examples/company-newsletter.md` - Para boletins informativos de toda a empresa
   - `examples/faq-answers.md` - Para responder a perguntas frequentes
   - `examples/general-comms.md` - Para qualquer outra coisa que não corresponda explicitamente a um dos acima
3. **Siga as instruções específicas** nesse arquivo para formatação, tom e coleta de conteúdo

Se o tipo de comunicação não corresponder a nenhuma diretriz existente, peça esclarecimentos ou mais contexto sobre o formato desejado.
