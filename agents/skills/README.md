# Índice de Habilidades (Skills)

Este diretório contém habilidades especializadas que ampliam as capacidades do agente. O agente deve consultar este índice para identificar qual habilidade utilizar para atender às solicitações do usuário de forma eficiente.

## Regras de Manutenção

> **REGRA MANDATÓRIA**: Sempre que uma nova skill for criada, removida ou tiver sua descrição alterada, o agente de IA **DEVE** atualizar este arquivo `README.md` imediatamente para refletir as mudanças. Mantenha este índice sempre sincronizado com o diretório `skills`.

## Habilidades Disponíveis

Abaixo está a lista de habilidades disponíveis e seus propósitos:

- **[algorithmic-art](./algorithmic-art/SKILL.md)**: Criação de arte algorítmica usando p5.js com aleatoriedade semeada e exploração interativa de parâmetros.
- **[brand-guidelines](./brand-guidelines/SKILL.md)**: Aplica as cores e tipografia oficiais da marca Anthropic para garantir consistência visual em artefatos.
- **[canvas-design](./canvas-design/SKILL.md)**: Criação de arte visual e designs gráficos em documentos .png e .pdf baseados em filosofias de design.
- **[doc-coauthoring](./doc-coauthoring/SKILL.md)**: Guia os usuários através de um fluxo de trabalho estruturado para coautoria de documentação complexa.
- **[docx](./docx/SKILL.md)**: Ferramentas para criação, edição e análise abrangente de documentos Word (.docx).
- **[frontend-design](./frontend-design/SKILL.md)**: Criação de interfaces frontend distintas e de alta qualidade (sites, componentes, dashboards) com foco em design criativo.
- **[internal-comms](./internal-comms/SKILL.md)**: Recursos e modelos para escrever comunicações internas eficazes (boletins, atualizações de status, FAQs).
- **[mcp-builder](./mcp-builder/SKILL.md)**: Ferramentas para construir novos servidores MCP (Model Context Protocol).
- **[pdf](./pdf/SKILL.md)**: Kit de ferramentas para manipulação de PDF: extração de texto/tabelas, criação, mesclagem e divisão de documentos.
- **[pptx](./pptx/SKILL.md)**: Ferramentas para criação, edição e análise de apresentações PowerPoint (.pptx), incluindo conversão de HTML para slides.
- **[skill-creator](./skill-creator/SKILL.md)**: Meta-habilidade para criar novas habilidades. Analisa instruções e gera a estrutura de diretórios e arquivos `SKILL.md`.
- **[slack-gif-creator](./slack-gif-creator/SKILL.md)**: Criação de GIFs animados otimizados para uso no Slack.
- **[theme-factory](./theme-factory/SKILL.md)**: Fábrica de temas para criar identidades visuais consistentes para artefatos.
- **[web-artifacts-builder](./web-artifacts-builder/SKILL.md)**: Criação de artefatos web complexos e multi-componentes (React, Tailwind) com suporte a roteamento e gerenciamento de estado.
- **[webapp-testing](./webapp-testing/SKILL.md)**: Testes automatizados end-to-end para aplicações web, garantindo funcionalidade e qualidade visual.
- **[xlsx](./xlsx/SKILL.md)**: Ferramentas para criação, edição e análise de planilhas Excel, com suporte a fórmulas e análise de dados.
